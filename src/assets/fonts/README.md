本目录用于存放前端自托管字体文件（woff2）。

当前 UI 设计方案使用的字体：
- `Noto Sans SC`（正文）：`NotoSansSC-300.woff2` / `NotoSansSC-400.woff2` / `NotoSansSC-500.woff2` / `NotoSansSC-700.woff2`
- `Noto Serif SC`（标题/品牌）：`NotoSerifSC-400.woff2` / `NotoSerifSC-500.woff2` / `NotoSerifSC-600.woff2` / `NotoSerifSC-700.woff2`
- `JetBrains Mono`（代码）：`JetBrainsMono-400.woff2` / `JetBrainsMono-500.woff2` / `JetBrainsMono-700.woff2`

字体声明位于：`src/assets/styles/fonts.css`
